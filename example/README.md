# Example

Demo app for the **liquid_circle_floating_nav_bar** package.

## What it shows

- A `Scaffold` with **`extendBody: true`** so the body extends behind the floating nav bar and content can scroll under it.
- `LiquidCircleFloatingNavBar` with 5 items (Home, Search, Chat, Feed, Personal).
- Programmatic page change via `GlobalKey<LiquidCircleFloatingNavBarState>` and `setPage(1)`.

## Run

From the package root:

```bash
cd example && flutter run
```

Or from this directory:

```bash
flutter run
```
